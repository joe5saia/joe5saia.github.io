Event Studies Data

A simple Docker Container to scrape the BLS website with Python for macroeconomic data release dates and make them into a useable CSV file that you can then pipe into whatever.

Bayesian Hidden Markov Model MCMC

BHMMMCM! Julia Repository for estimating a three state hidden Markov chain model with Markov Chain Monte Carlo methods. The goal was to estimate how the model "learns" as new data comes in each month, which we approximated by estimating the model on a rolling window of data spanning 50 years. To best utilize the HPC clusters available to us, the project is set up to spawn several hounded copies of the core estimation code and run in parallel over the rolling window. The fancier versions included an additional estimation step of treating the observations as noisy observations that need to be filtered.

Fedplots

Rewrite of the Federal Reserve Board of Governors Splus plotting package into R, modernizing the code base, increasing ease of use, performance and capabilities while maintaining visual continuity, currently used by the majority of research assistants at the Board of Governors. The code is locked away, deep in the Federal Reserve's computer vaults, but output is available at the link.
